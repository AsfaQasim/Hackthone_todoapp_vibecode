# Todo App Frontend

This is the frontend for the Todo application using Next.js and a simple authentication system.

## Setup Instructions

1. Make sure you have Node.js installed on your system.

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

   Or directly using:
   ```bash
   npx next dev
   ```

The application will be available at http://localhost:3000 (or another available port if 3000 is in use).

## Project Structure

- `app/` - Contains the Next.js 16+ app router pages
- `components/` - Reusable React components
- `lib/` - Utility functions
- `public/` - Static assets

## Key Features

- Simple authentication system with login/signup
- Modern Next.js 16+ with App Router and Turbopack
